# Marcus 三重過濾法 + Wenty 成交量與訂單流分析

## Michael Marcus「三重過濾法」

### 核心原則

Marcus 的交易哲學專注於：只在高確定性機會下重倉，透過多層過濾剔除低品質標的。

### 第一重：基本面過濾（供需失衡）

- 尋找宏觀層面出現供需失衡的標的
- 關鍵問題：**這個預期是否已被市場計價？**
- 若市場尚未察覺，該標的才具備交易價值
- 實作指標：
  - 營收年增率突變
  - 產業政策變化
  - 國際事件驅動（關稅、匯率、原物料）
  - 法人買超集中度

### 第二重：市場情緒過濾（籌碼博弈）

- 觀察市場對消息的反應
- **利空不跌** = 空頭已平倉、多頭在接盤 = 強烈看多訊號
- **利好不漲** = 多頭已出場 = 看空訊號
- 實作指標：
  - 三大法人買賣超方向
  - 融資增減（散戶反向指標）
  - 券商分點進出
  - 週轉率異常

### 第三重：避開被套買家

- 優先選擇**順應趨勢**的標的
- 避開那些價格漲不動、買家被困在高位的逆勢標的
- 實作指標：
  - 均線排列（5>20>60 多頭排列）
  - 距離近期高點的距離
  - 套牢籌碼比例（均線糾結度）

## Fabio Wenty「成交量與訂單流分析」

### 核心技術 1：CVD（Cumulative Volume Delta）背離分析

**定義**：CVD = 累積（主動買量 - 主動賣量）

**台股代理指標**（因盤後無 tick 資料）：
- 日 K 代理：CVD_proxy = (close - open) / (high - low + 1e-6) * volume
- 或用三大法人買賣超作為機構 CVD 代理

**背離判斷**：
- 價格創新高 + CVD 同步創新高 → 健康上漲，繼續持有
- 價格創新高 + CVD 走平或下跌 → 買盤後繼無力，謹慎/減倉
- 價格創新低 + CVD 不再創新低 → 賣壓衰竭，可能反轉

### 核心技術 2：吸收現象（Absorption）

**定義**：在關鍵價位，大量買單湧入但價格漲不動 → 買單被賣方掛單「吸收」

**偵測方法**：
- 成交量急增 + 價格波動收窄
- 具體指標：high - low 收窄 + volume 放大
- 吸收比 = volume / (high - low + 1e-6)
- 吸收比突然放大 → 可能是趨勢反轉前兆

### 核心技術 3：衝浪手技術（Surfer Entry）

**定義**：在關鍵日內價位精準切入

**關鍵價位**：
- 前一日收盤價
- 前一日最高/最低價
- 整數關卡（百元、千元）
- 近期支撐/壓力線

**切入判斷**：
- 判斷正確 → 價格立即脫離成本區
- 價格左右橫跳 → 時機未到，撤離
- 觸及極窄止損（10-15 點）→ 立即撤離

**台股日 K 代理**：
- 無法做日內切入，但可用「隔日開盤位置」+「前日關鍵價位」判斷
- 開盤價 > 前日高點 + 量能確認 = 衝浪手信號

## 四階段篩選流程（整合版）

```
全台股 1000+ 檔
    ↓
Stage 1: 宏觀與情緒過濾
    ├─ 基本面未計價 (營收突變/政策/事件驅動)
    ├─ 籌碼博弈 (利空不跌 / 法人買超)
    └─ 順勢過濾 (均線多頭排列)
    → 篩出 50-100 檔
    ↓
Stage 2: 技術形態確認
    ├─ 五日高點突破
    ├─ 放量結構突破 (量增 > 1.5 倍 20 日均量)
    └─ 收盤站穩阻力位上方
    → 篩出 10-20 檔
    ↓
Stage 3: 微觀精準點位
    ├─ CVD 背離確認 (健康上漲 vs 買盤無力)
    ├─ 吸收現象偵測 (關鍵位大量不漲)
    └─ 衝浪手切入位 (前日關鍵價位)
    → 鎖定 3-5 檔
    ↓
Stage 4: 動態風險管理
    ├─ ABC 機會分級
    ├─ 金字塔加倉 (只在賺錢時加)
    └─ 移動止損 (五日高點滾動)
    → 最終交易名單
```

## 3-7 天交易週期適配

### 時間框架

- 入場信號：日 K 級別（五日高點突破 = 5 天窗口）
- 持倉週期：3-7 個交易日
- 出場信號：
  - 移動止損觸發（跟隨五日高點下移）
  - CVD 背離出現
  - 吸收現象確認
  - 基本面邏輯消失

### 進場條件清單

1. ✅ Stage 1 通過（至少 2/3 子條件）
2. ✅ Stage 2 通過（五日高點突破 + 放量）
3. ✅ Stage 3 CVD 確認（無背離）
4. ✅ 無吸收現象（未被大單阻擋）
5. ✅ 開盤位置在衝浪手切入區

### 出場條件清單

1. 收盤跌破五日高點（移動止損）
2. CVD 背離出現（價格新高 + CVD 走平/下降）
3. 吸收現象確認（大量不漲）
4. 持倉滿 7 天（時間止損）
5. 基本面邏輯消失
