## ⚠️ 陷阱 22：技術要點品質標準（必讀）

### ❌ 錯誤示範 — 流水線式項目標題（無洞見、無價值）

```
1. 解決的問題：提升透射率
2. 核心發明：改善對比度（contrast ratio）
3. 關鍵技術特徵：維持電壓保持率（VHR）
4. 實施方式：包含 Formula I 化合物
5. 與先前技術的差異：優於先前技術
```

**問題**：每項僅 3-8 字，只是關鍵詞列舉，讀者無法理解技術本質、參數矛盾或創新機制。

### ✅ 正確示範 — 融會理解的判斷性洞見（有價值）

```
1. 解決的問題：現有 VA 模式液晶介質在追求高 |Δε| 時，常伴隨旋轉粘度 γ1 上升及
   低溫穩定性下降的取捨困境。先前技術中負介電與低粘度不可兼得，成為高對比度
   顯示器開發的核心瓶頸。

2. 核心發明：本發明通過將含 CF₂O/OCF₂ 連接基的式 I 化合物與特定共混組分
   （式 IIA-IID）組合，在維持 |Δε|≥3.0 的同時將 γ1 控制在 100 mPa·s 以下，
   並確保 −20°C 低溫下無結晶析出，突破了先前技術的瓶頸。

3. 關鍵技術特徵：從屬項進一步限定式 I 化合物選自式 I-1 至 I-17 的具體化合物群，
   烷基限定為 C1-C6 直鏈烷基，此收窄範圍使端基取代基精確調控介電異向性與旋轉
   粘度等關鍵參數，而非泛用結構。

4. 實施方式：[未提取到實施例段落，無法判斷] 具體化合物配方、組成物比例及
   性能參數未於提取內容中提供。

5. 與先前技術的差異：相較於先前技術中單一類型化合物無法同時達成高負 Δε 與
   低 γ1 的限制，本發明通過式 I + 式 IIA-IID 組合實現綜合性能平衡，特別是
   改善響應時間以實現節能顯示，是先前技術中未見的組合策略。
```

**關鍵差異**：每項 30+ 字連貫論述，指出參數矛盾本質（高 |Δε| vs 低 γ1）、創新機制（CF₂O 連接基 + 共混組合）、限定邏輯（收窄範圍的原因），而非僅列舉關鍵詞。

### 品質檢查清單

## US20240067879A1

1. 解決的問題：現有液晶顯示模式（如TN模式）存在對比度視角依賴性強的缺陷；VA模式雖改善視角但仍有均勻性問題；FFS及UB-FFS模式需具備高負介電異向性（負Δε）與高可靠性的液晶介質，然先前技術的材料在負Δε、相位範圍、Δn及長期穩定性（LTS）等關鍵參數上難以兼顧，無法同時滿足低響應時間與節能顯示的需求。

2. 核心發明：本發明提供一種包含兩大組分的液晶介質：a)一種或多種式I化合物，其特徵在於含CF₂O、OCF₂等連接基及F/Cl鹵素取代基，具有高負Δε；b)一種或多選自式IIA、IIB、IIC及IID的化合物作為共混組分。該組合使液晶介質兼具適當高負Δε、合適相位範圍與Δn以及高LTS，顯著降低先前技術的缺點，實現改善的響應時間以支援節能顯示器。

3. 關鍵技術特徵：從屬項進一步限定式I化合物選自式I-1至I-17的具體化合物群，其中烷基與烷基*各自獨立表示含1-6個碳原子的直鏈烷基，烯基表示含2-6個碳原子的直鏈烯基，環烷基表示含1-10個碳原子的直鏈或支鏈烷基且其中一個CH₂基團被取代，進一步收窄了化合物的結構範圍。

4. 實施方式：[未提取到實施例段落，無法判斷] 具體化合物配方、組成物比例及性能參數未於提取內容中提供。

5. 與先前技術的差異：相較於先前技術的液晶材料，本發明透過將式I含鹵素及CF₂O/OCF₂連接基的化合物與式IIA-IID化合物組合，在維持適當高負Δε與合適相位範圍及Δn的條件下，實現了更高的LTS，使材料缺點顯著降低，特別是改善了響應時間以實現節能顯示，這是先前技術中單一類型化合物難以同時達成的綜合性能。

## US20250207032A1

1. 解決的問題：現有液晶顯示器（特別是IPS及FFS模式）所用之介電正性液晶介質，在旋轉黏度（γ1）與彈性常數（K22、K11）之比值偏高，導致響應時間較慢；此外，現有介質的平均彈性常數（Kav）不足，使散射參數較大，暗態表現不佳而降低對比度。同時，低溫穩定性不足亦限制其在車載等行動應用之適用性。
2. 核心發明：本發明提供一種具正介電各向異性之液晶介質，其特徵在於組合使用式I化合物與式II及/或式III化合物。該組合使介質具備低旋轉黏度及低γ1/K22與γ1/K11比值，從而實現HB-FFS、FFS及IPS顯示器之高亮度、高穿透率及快速響應。同時可在維持其他物理性質下提升Kav，降低散射參數以改善暗態與對比度。
3. 關鍵技術特徵：依據第2項請求項之進一步限定，式I化合物中之L11與L12各自獨立地表示H或F，此為核心發明中式I化合物端基取代基的具體優選範圍，有助於精確定義液晶分子結構並調控其介電與光學特性。
4. 實施方式：[未提取到實施例段落，無法判斷] 具體化合物組成、配方比例及性能參數數據因實施例段落未提取到而無法提供。
5. 與先前技術的差異：相較於先前IPS與FFS顯示器所用之液晶材料，本發明透過式I、II、III化合物的特定組合，同時實現低旋轉黏度與高平均彈性常數，突破既有材料無法兼顧快速響應與高對比度之瓶頸。此外，本發明介質具備優異的低溫穩定性，可穩定運作於低至-40°C，顯著優於先前技術，特別適用於車載等行動應用場景。

## 方法論備註

- **生成後端**: hermes_agent（Hermes Agent 直接讀取 prompt 並生成）
- **分段策略**: 三層回退 — 標題行匹配 → 啟發式分段（過渡段特徵詞）→ 段落號回退
- **Fallback 機制**: 獨立進程 `extract_patent_sections()` 失敗時，從已有數據組裝 prompt
- **嚴禁編造規則**: 未提取到的段落（如 Examples=0）標註 `[未提取到實施例段落，無法判斷]`
- **持久化路徑**: 
  1. E2E 腳本生成 prompt → `reports/tech_feature_prompts_batch.json`
  2. Hermes Agent 生成技術要點 → `reports/tech_features_<PATENT_ID>.json`
  3. E2E 腳本再次執行時自動回填 → `p['tech_features']`
