# arxiv

Skill for searching and retrieving academic papers from arXiv.

## Description

This skill provides tools to search arXiv for academic papers, retrieve paper details, and download paper content.

## Usage

Use this skill when you need to:
- Search for papers by topic, author, or title
- Retrieve abstracts and metadata
- Download PDF or source files

## Commands

The skill integrates with arXiv's API to provide paper search and retrieval capabilities.

## Notes

- arXiv IDs follow the format: arXiv:YYMM.NNNNN or arXiv:category/YYMMNNNNN
- Results are limited to recent and relevant papers
- Always cite arXiv papers properly when referencing them
